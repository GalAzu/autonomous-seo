💡 מדריך Robots.txt
User-agent: * — חל על כל הסורקים
Allow: / — אפשרו גישה לכל העמודים
Disallow: /admin/ — חסימת נתיב
Sitemap: https://example.com/sitemap.xml — הפניה למפת האתר

אסטרטגיית מילות מפתח

מיפוי מילות מפתח לעמודים — יעדי תוכן לדירוג בגוגל

💡 חשוב לדעת
גוגל מתעלם מתגית <meta keywords> מאז 2009. מילות המפתח כאן הן יעדי תוכן — שלבו אותן בכותרות, H1, H2, גוף הטקסט וטקסט alt בתמונות.

ניתוח מבנה כותרות

היררכיית H1-H6 לכל עמוד — קריטי ל-SEO. כל עמוד צריך בדיוק H1 אחד.

📖 מדריך הגדרת GSC▼

1. אימות: GSC → הוספת נכס → URL Prefix → הזינו:

2. מפת אתר: GSC → Sitemaps → הזינו:

3. אינדוקס: GSC → URL Inspection → הזינו כתובת → Request Indexing

4. ביצועים: GSC → Performance → Export → הדביקו כאן

טיפים לאופטימיזציה:

• הרבה חשיפות + CTR נמוך = שכתבו כותרת ותיאור

• מיקום 5-20 = קרובים לעמוד 1, הוסיפו תוכן

• שאילתות חדשות = הוסיפו אותן לתוכן
